# Okugai Cafe ☕️

Sebuah *landing page* responsif dan modern untuk Okugai Cafe, Majalaya. Dibuat dengan antarmuka yang elegan menggunakan kombinasi HTML5, CSS3 murni (Vanilla CSS) dengan gaya *glassmorphism*, serta JavaScript untuk animasi dan interaksi.

## 🌟 Fitur Utama

*   **Desain Premium & Modern**: Tampilan web yang menggunakan palet warna kopi dengan komponen transparan yang terkesan bersih dan elegan.
*   **Responsif**: Sepenuhnya dioptimalkan untuk perangkat seluler, tablet, dan desktop.
*   **Smooth Scrolling & Animasi**: Navigasi mulus ke berbagai bagian halaman, dilengkapi dengan efek *fade-in* (*Intersection Observer*) saat melakukan *scroll*.
*   **Indikator Status Buka**: JavaScript memantau jam untuk menampilkan status "Buka" atau "Tutup" secara dinamis berdasarkan jam tayang.
*   **Tanpa Framework Eksternal**: Dibangun murni dengan Vanilla HTML, CSS, dan JS, memastikan latensi *loading* yang super cepat – sangat ideal untuk langsung di-deploy misalnya pada Vercel.

## 🛠 Teknologi yang Digunakan

*   **HTML5**: Struktur halaman semantik.
*   **CSS3**: Flexbox, CSS Grid, Media Queries, CSS Variables, Animations (`@keyframes`), dan efek tipe *Glassmorphism*.
*   **JavaScript (ES6+)**: Interaksi DOM dan *Intersection Observor* untuk pemicu visibilitas elemen.
*   **Remix Icons**: Ikon yang modern dan tajam.
*   **Google Fonts**: Menggunakan font 'Outfit'.

## 🚀 Instalasi & Deployment

Proyek ini tidak memerlukan proses *build* atau instalasi *dependency*.

### Menjalankan secara Lokal

1.  *Clone repository* ini:
    ```bash
    git clone https://github.com/arsyadal/okugaicafe.git
    cd okugaicafe
    ```
2.  Buka file `index.html` secara langsung di browser Anda, atau gunakan *Live Server* di VS Code untuk pengalaman pengembangan yang lebih baik.

### Deployment (Contoh Vercel)

Karena ini situs statis murni:
1. Hubungkan repositori GitHub ini ke Vercel *Dashboard*.
2. Vercel akan otomatis mendeteksi bahwa ini adalah situs statis.
3. Klik "Deploy", dan *website* siap dikunjungi dalam hitungan detik.
